Pré-requisitos

Como o Software atualmente está em desenvolvimento e não há uma versão de build que funcione corretamente, você precisará ter acesso aos itens abaixo:

  • Unity Engine: Você precisará ter instalado em sua máquina o Unity Engine para executar o jogo, pois como não há uma build funcional para o jogo de fato, você tem que executar ele no modo de desenvolvimento.
  • Webcam: O jogo usa inteligência Artificial como Input dentro do jogo para que você possa mover as peças dentro dele, por conta disso você precisara de uma webcam conectada em sua máquina para que você possa jogar. E lembre-se, sempre que for jogar sempre utilize um local com boa iluminação
